# Calendify Pro

**Calendify Pro** es una solución de agendamiento inteligente diseñada para negocios locales como estéticas, spas, barberías y más. Este sistema permite a los negocios gestionar fácilmente citas, clientes y servicios desde una interfaz moderna e intuitiva.

## ✨ Características destacadas

- Registro y login para usuarios y negocios
- Panel de control personalizado para cada tipo de usuario
- Registro de negocios con ubicación en el mapa
- Mapa interactivo para clientes con negocios cercanos
- Opción de tarjeta de lealtad para fidelización de clientes
- Diseño profesional y responsivo inspirado en Smartlink Portal
- Íconos visuales y navegación moderna
- Landing page estilo SaaS con animaciones

## 🚀 ¿Cómo usar?

1. Descomprime este paquete
2. Sube todos los archivos a un repositorio en GitHub
3. Activa GitHub Pages en la pestaña Settings > Pages
4. Tu sitio estará disponible en: `https://<tu-usuario>.github.io/calendifypro/`

## 📂 Estructura de carpetas

- `index.html` — Página principal
- `login.html` — Elección de tipo de usuario
- `login-user.html`, `login-admin.html` — Login para clientes y negocios
- `registro-usuario.html`, `registro-negocio.html` — Formularios de registro
- `user-panel.html`, `admin.html` — Paneles diferenciados
- `mapa.html` — Vista de negocios en mapa
- `styles.css` — Estilos visuales personalizados
- `README.md` — Documentación general
